# 通用控制器需求文档 - 非功能需求

> 版本: v3.17

## 1. 性能需求

| 指标 | 要求 | 备注 |
|------|------|------|
| 控制频率 | ≥ 50 Hz | |
| MPC 求解时间 | ≤ 10 ms (P95) | ACADOS 优化 |
| 端到端延迟 | ≤ 15 ms | |
| CPU 占用 | ≤ 20% (单核) | |
| 状态估计频率 | ≥ 100 Hz | EKF |
| 控制器切换时间 | ≤ 20 ms | 热备 |
| 平滑过渡时间 | 可配置 100-500 ms | |

## 2. 可靠性需求

| 指标 | 要求 |
|------|------|
| 可用性 | 99.9% |
| 故障恢复时间 | ≤ 50 ms |
| 无崩溃运行时间 | ≥ 24 小时 |
| 热备切换时间 | ≤ 20 ms |
| 预测性降级准确率 | ≥ 90% |
